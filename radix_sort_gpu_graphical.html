1. block histogram：每个 workgroup / block 先做局部统计

2. global prefix / scan：把局部 histogram 扫成全局偏移

3. parallel scatter：每个元素计算自己的目标位置

优化点

局部 histogram
每个 block 只统计自己负责的元素，减少全局原子操作压力。
prefix scan
对 bucket × block 的计数做扫描，得到每个 block 在每个 bucket 的全局起点。
稳定 scatter
位置 = bucket 全局起点 + block 偏移 + block 内同 digit 的局部 rank。
ping-pong buffer
每轮从 input 读、向 output 写，然后交换指针，避免原地搬移冲突。
